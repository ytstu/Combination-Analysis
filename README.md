# 组合装数据处理

## 数据说明

- `data/input/模拟-i输入-讲解.xlsx`
    - 输入文件。
    - 必须包含 `原始商品编码` 列。
    - 程序按 `单品编码*倍数` 的格式解析，例如 `A123*3`。

- `data/input/商品资料.xlsx`
    - 单品资料库。
    - 程序使用 `商品编码` 列判断拆分后的单品编码是否已存在。

- `data/input/组合资料.xlsx`
    - 组合装资料库。
    - 程序使用 `组合商品编码` 列判断原始组合编码是否已存在。

- `data/output/组合装数据MMDD.xlsx`
    - 程序输出文件，文件名中的 `MMDD` 为运行当天日期。
    - 输出列为：`组合商品编码`、`商品编码`、`数量`。

## 程序使用方式

1. 将输入文件放在 `data/input/` 目录下，并保持以下文件名：
    - `模拟-i输入-讲解.xlsx`
    - `商品资料.xlsx`
    - `组合资料.xlsx`
2. 在项目根目录执行：

```bash
python3 src/main.py
```
